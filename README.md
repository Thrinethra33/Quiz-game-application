# Quiz Game Application

This repository contains the solution for **Task 03** of the SkillCraft Technology internship, as outlined in the task guidelines from IMG_8613.jpg. The project is an interactive, web-based quiz application designed to test user knowledge across a variety of question structures and provide instant performance feedback.

## 🚀 Project Features

* **Dynamic Question Rendering:** Seamlessly displays diverse question types, including Single-Select (Multiple Choice), Multi-Select, and Fill-in-the-Blanks.
* **Interactive Gameplay:** Captures user answers, handles progression from question to question, and includes visual feedback for selected answers.
* **Automated Scoring System:** Evaluates correctness on the fly and presents a summary of the final score upon completion of the quiz.
* **Responsive UI:** Clean, distraction-free interface built to scale comfortably across mobile devices, tablets, and desktop computers.

## 🛠️ Technologies Used

* **HTML5:** Provides the semantic foundation for the quiz canvas, option inputs, and submission buttons.
* **CSS3:** Delivers styling for layout structures, transitions between states, and active selection feedback.
* **JavaScript (ES6):** Controls the game flow logic, validates user inputs, parses data structures for questions, and manages the scoring engine.
